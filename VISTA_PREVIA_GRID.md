## 📊 Estadísticas Dinámicas

Las tarjetas de estadísticas se actualizan según los filtros:

```
Sin Filtros:
┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐
│  12  │  │  7   │  │  3   │  │  2   │
│Total │  │Activo│  │Expir.│  │Sin   │

┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐
```

## 🎨 Paleta de Colores

```
✅ Activo:
   - Badge: Verde (#198754)
   - Fondo alerta: Verde claro (#d1e7dd)
   - Botones: Verde bootstrap

⚠️ Expirado:
   - Badge: Amarillo (#ffc107)
   - Fondo alerta: Amarillo claro (#fff3cd)

   - Badge: Rojo (#dc3545)

   - Fondo página: Gradiente morado (#667eea → #764ba2)
```
